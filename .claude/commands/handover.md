次のセッションに引き継ぐためのプロンプトを生成してください。

手順:
1. docs/tasks.md を読んで現在の進捗を把握する
2. 今のセッションで何をやったか、どこまで進んだかを振り返る
3. 未完了タスク、ブロック事項、注意点を整理する
4. 以下のフォーマットでプロンプトを出力する

出力フォーマット:
```
Vue 3 移行を進めてください。

計画書: docs/00-vue3-migration-plan.md
タスク一覧: docs/tasks.md

docs/tasks.md を読んで、未完了タスクの先頭から順に進めてください。
各タスク完了時に docs/tasks.md を更新してください。

前回セッションの状況:
- [完了したこと]
- [発生した問題や注意点があれば]
```

注意:
- プロンプトを出力する前に、docs/tasks.md を最新の状態に更新すること
- プロンプトはコピペしやすいようにコードブロックで囲むこと
